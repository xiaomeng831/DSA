## 1.HashTable(哈希表)
### 1.说明
1. Python
    - 在Python中, 哈希表指的是字典 `dict`
2. Java
    - 在Java中, 哈希表指的是 `Map<k, v>接口` 的实现类 `HashMap<k, v>`
    - Java中有一个类 `Hashtable<K, V>`, 是Java早期引入的一个类, 因为有诸多缺点（性能较低, 多线程环境中同步开销等）, 现在几乎被 `HashMap<k, v>` 取代
## 2.Set(集合)
### 1.说明
1. Python
    - 在Python中, 集合指的是 `set`
2. Java
    - 在Java中, 集合指的是 `Collection<E>接口` 的子接口 `Set<E>` 的实现类 `HashSet<E>`
